                    Hummingbot Lite - 项目总结

================================================================================
                        技术栈
================================================================================

语言：Python 3.12+
Web 框架：FastAPI
实时通信：WebSocket
交易所 API：ccxt
配置管理：PyYAML
日志系统：colorlog

================================================================================
                        API 端点
================================================================================

GET  /                    - Web 控制面板
GET  /api/status          - 获取机器人状态
GET  /api/balance         - 获取账户余额
GET  /api/orders          - 获取订单列表
POST /api/start           - 启动策略
POST /api/stop            - 停止策略
POST /api/cancel-all-orders - 取消所有订单
GET  /api/events          - 获取事件历史
GET  /api/performance     - 获取策略表现
WS   /ws                  - WebSocket 实时通信

================================================================================
                        使用流程
================================================================================

1. 安装依赖
   ./install.sh

2. 配置 API 密钥
   编辑 config.yaml，填写 OKX API 信息

3. 启动程序
   ./start.sh

4. 访问控制面板
   http://localhost:5000

5. 启动策略
   在 Web 控制面板点击"启动"按钮

6. 监控运行
   实时查看策略状态、仓位、盈亏

7. 停止策略
   在 Web 控制面板点击"停止"按钮
